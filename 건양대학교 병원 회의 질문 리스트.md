#### 질문 1: 병원 내 의료 정보 시스템 구조 및 연동 방식

**질문:**
건양대학교 병원에서 현재 사용 중인 의료 정보 시스템(EMR, OCS, RIS, LIS, PACS)의 구조와 시스템 간 연동 방식은 어떻게 되나요?

**세부 질문:**
- 각 시스템이 독립적으로 운영되는지, 통합 시스템인지?
- 시스템 간 데이터 교환 방식 (API, 메시지 큐, 직접 DB 접근 등)
- FHIR 표준을 사용하고 계신가요?
- 외부 시스템(다른 병원, 보험사)과의 데이터 교환 방식

**이유:**
프로젝트에서 EMR, OCS, RIS, LIS, Orthanc(DICOM 서버)를 구현해야 하는데, 실제 병원 시스템 구조를 참고하여 설계해야 함

#### 질문 4: 멀티모달 데이터 결합 (영상 + EMR 정보)

**질문:**
현재 저희 계획은 MRI 영상과 유전체 데이터 분석에 집중되어 있습니다. 하지만 교수님의 여러 선행 연구들을 보면 환자의 건강 행태나 주관적 건강 인식 같은 임상 정보들도 결과에 큰 영향을 미치는 것 같습니다. 저희 CDSS의 예측 성능을 높이기 위해, 영상 데이터 외에 EMR상의 어떤 환자 특성 정보(나이, BMI, 기저질환 등)를 추가 변수로 결합(Fusion)하는 것이 가장 효과적이라고 보시는지 궁금합니다.

**세부 질문:**
- 영상 데이터와 결합 시 효과적인 EMR 변수
- 나이, BMI, 기저질환 등의 중요도
- 데이터 결합 방법 (Early Fusion, Late Fusion 등)
- 성능 향상 기대치

## 3. 구조 이상 및 비대칭 (Architectural/Asymmetry) 표시 방법

### 특징
- 종괴나 석회화처럼 명확한 병변이 아님
- 유방 조직의 구조적 변화 또는 좌우 비대칭
- 양성일 수도 있지만 추가 검사 필요

### UI 표시 방법

#### 1. 아이콘 + 색상
```
🟠 Architectural/Asymmetry
```
- 주황색 (경고와 정상 사이)
- 정보 아이콘 (ℹ️) 또는 구조 아이콘 (🏗️)

#### 2. 상세 설명 툴팁
```
┌─────────────────────────────────────────┐
│ 🟠 구조 이상 및 비대칭                   │
│                                          │
│ 유방 조직의 구조적 변화 또는            │
│ 좌우 비대칭이 관찰되었습니다.           │
│                                          │
│ 이것은:                                  │
│ • 양성 변화일 수 있습니다               │
│ • 추가 영상 검사가 필요할 수 있습니다   │
│ • 이전 검사와 비교가 중요합니다         │
│                                          │
│ 권장: 전문의 상담                        │
└─────────────────────────────────────────┘
```

#### 3. 비교 뷰 (좌우 대칭 비교)
```
┌──────────────────────────────────────────────────┐
│  좌측 유방          vs          우측 유방         │
│  ┌──────────┐               ┌──────────┐        │
│  │ L-CC     │               │ R-CC     │        │
│  │ [이미지] │               │ [이미지] │        │
│  │          │               │          │        │
│  │ 🟢 정상  │               │ 🟠 비대칭 │        │
│  └──────────┘               └──────────┘        │
│                                                   │
│  💡 우측 유방에서 구조적 비대칭이 관찰됩니다.    │
│     좌측과 비교하여 조직 밀도 차이가 있습니다.   │
└──────────────────────────────────────────────────┘
```

## 6. 구현 우선순위

### Phase 1 (필수)
1. ✅ 4장 이미지 그리드 레이아웃
2. ✅ 각 이미지별 AI 분석 버튼
3. ✅ 결과 오버레이 (색상 코딩)
4. ✅ 확률 표시

### Phase 2 (권장)
5. 📊 확률 그래프
6. 💡 해석 및 권장사항
7. 📄 보고서 다운로드

### Phase 3 (선택)
8. 🔄 좌우 비교 뷰
9. 📅 타임라인 (이전 검사 비교)
10. 🔔 알림 및 전문의 전송
